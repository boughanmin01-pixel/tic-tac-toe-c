# Tic-Tac-Toe en langage C

Jeu de morpion développé en langage C en mode console.

## Fonctionnalités
- Affichage du plateau
- Alternance des joueurs
- Validation des entrées
- Détection victoire / match nul

## Technologies
- Langage C

## Exécution
Compiler avec un compilateur C (ex: GCC) puis exécuter dans la console.
